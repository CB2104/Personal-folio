Sobre mi


Soy César Bastidas, un desarrollador front-end en formación basado en Buenos Aires. A través de aprendizaje autodidacta, he construido una base sólida en HTML, CSS y JavaScript, practicando con proyectos como apps de clima, quizzes y listas de tareas que involucran manipulación del DOM, eventos y lógica asincrónica.

Actualmente, estoy explorando React para crear componentes reutilizables y Tailwind CSS para diseños eficientes. Mi enfoque es en código limpio, responsivo y centrado en el usuario, mientras resuelvo problemas de manera proactiva. Disfruto el proceso de aprendizaje continuo y busco oportunidades junior donde pueda contribuir con soluciones prácticas. A mediano y largo plazo, quiero internarme también en React Native y trabajar con herramientas de línea de comandos (CLI), para diversificar mi perfil hacia lo móvil y utilitarios.

Si te interesa mi trabajo, revisa mis proyectos o contáctame para charlar sobre colaboraciones.
